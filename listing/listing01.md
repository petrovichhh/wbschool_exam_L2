Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
Выведет срез от 1 первого индекса до 3 включительно.
Передаем в массив интов b с 1 по 3 элементы массиыа a.
Вывод: 77, 78, 79.
