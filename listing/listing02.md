Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}


Ответ:
Функция test() вернет 2. Вывод: 2.
Порядок такой:
Сначала x = 1. После чего выполнится отлеженная функция инкремента x. 
defer func() {
		x++
	}()

Функция anotherTest() вернет 1. Вывод 1.
Порядок такой:
Сначала инициализируется переменная x, var x int. После чего выполнится присвоение x = 1. 
Вернется 1. Отложенная функция сработает только после return x, поэтому результат 1.
Ключевое слово defer используется для отложенного выполнения функции. Если есть множество defer, то порядок выполнения LIFO(Последним пришел, первым ушел).
